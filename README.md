# Catálogo de Filmes com TMDB

**Nome:** Maryanne Ilha da Luz

## Endpoint utilizado

Foi utilizado o endpoint de filmes populares da API do TMDB

## Descrição

O sistema realiza uma requisição à API do TMDB utilizando Fetch API. Os dados recebidos são tratados em JavaScript e renderizados dinamicamente em cards contendo poster, título, ano, nota e sinopse dos filmes.
